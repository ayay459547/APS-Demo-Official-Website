# APS_CORE 先進排程系統 (Advanced Planning & Scheduling)

### 驅動高效智造的決策核心 (SMART SCHEDULING) > 基於雲端原生架構與先進 AI 演算法，為複雜製造環境提供即時、精準的產能規劃。將廠區數據轉化為實質的生產競爭力。

🌟 核心生態系架構 (Dual-Track Ecosystem)

1. 全域指揮中心 (Global Command - Web 版)

專為生管人員與高階決策者打造的高算力排程中樞。

全域可視化：甘特圖、機台負荷與即時 OEE 監控。

策略沙盤推演：支援無限版本的排程快照 (Snapshot) 儲存與多指標並排對比。

2. 行動協同終端 (Mobile Edge - App 版)

專為生產現場設計的敏捷終端，確保執行精準與即時回報。

單手直覺操作：符合現場情境的 UI/UX 設計，極致簡化操作流程。

異常即時響應：機台異常、物料短缺一鍵回報，毫秒級同步至全域指揮中心。

智慧換線建議：現場可直接依據系統建議執行換線排程，動態調整不受限。

🚀 系統核心亮點 (Core Features)

🧠 AI 驅動的動態排程引擎 (Agent Loop)

告別經驗法則與繁瑣的人工試算。導入具備最佳化演算法的 AI 排程助理，透過標準化的決策迴圈，即時偵測產線瓶頸、動態推薦最佳權重配置。

多維度限制解析：精準對接 ERP/MES，全面解析 BOM、途程 (Routing) 與設備能力。

多目標最佳化：在訂單達交率 (OTD)、設備稼動率 (OEE) 與換線成本之間取得完美平衡。

⚖️ 多版本情境模擬與分析 (What-If Simulation)

降低決策風險的關鍵。系統支援無限版本的排程快照與假設性情境模擬。透過多維度對比雷達圖與優劣勢洞察，落實基於數據的決策 (Data-Driven Decision Making)。

⚡ 雲端原生架構，極致運算效能

採用微服務架構與 WebAssembly 邊緣運算技術，將複雜的基因演算法與龐大的數據渲染無縫卸載。無需安裝厚重軟體，您的瀏覽器即是具備極致效能的排程工作站。
